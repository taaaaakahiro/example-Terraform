# terraform-practice

### SetUp
 1. setup network
    ```
    $ cd ./dev/network
    $ terraform plan
    $ terraform apply  
    ```
2. setup modules
    ```
    $ cd ./dev/services
    $ terraform plan
    $ terraform apply  
    ```
